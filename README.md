# Tentacle-Tactics
##  Controls
           -  Shoot = Sapce
           -  Move = Arrow Keys(Left and Right)
           
           
        
### Objective
         Defend from
         attacking tenticle 
         creatures.
         
         Do not get touched
         by the enemy creature.
         
         You are given 5 
         hits before your 
         ship's defense shield
         gives out.
